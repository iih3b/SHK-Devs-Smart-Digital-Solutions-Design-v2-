## 📌 About The Project

This is the **second design version** of the SHK Devs course project.
A fully redesigned take on the same fictional digital agency, featuring a more dynamic two-column hero layout, animated orbital rings around the logo, and Font Awesome icons throughout.

Built as part of an **HTML course assignment**.

> ⚠️ **Note:** SHK Devs is a fictional concept created purely for academic purposes. All clients, statistics, and services shown are illustrative only.

---

## ✨ Design Highlights

- 🌀 **Animated hero rings** — two spinning orbital rings around the floating logo image with glow blur
- 📐 **Two-column hero layout** — text copy on the left, animated visual on the right
- 🎨 **Cyan `#00d4ff` / Blue `#3b5bdb` / Violet `#7c3aed`** color system via CSS variables
- 🔲 **CSS grid background** — subtle animated mesh overlaid across the full page
- 🟢 **Pulsing dot badge** — animated live-indicator in the hero eyebrow label
- 📱 **Full-screen overlay mobile menu** — large display-font links, smooth open/close
- 🧭 **Transparent → frosted glass navbar** — transitions in on scroll
- 🌟 **Scroll reveal system** — `.reveal` + `.reveal-delay-*` classes with IntersectionObserver
- 🏆 **Achievements grid** — 16 compact stat cards in the About section
- 🎭 **Team card gradient ring** — appears on avatar on card hover
